При этом пользователю не требуется выполнять дополнительных действий для входа.</span><span class="sxs-lookup"><span data-stu-id="3a611-117">The SSO token can be used by your backend to retrieve an access token for Microsoft Graph without the user having to do any additional sign in action.</span></span>
- <span data-ttu-id="3a611-118">Маркер единого входа предоставляет больше сведений об удостоверении, например отображаемое имя пользователя.</span><span class="sxs-lookup"><span data-stu-id="3a611-118">The SSO token provides richer identity information, such as the user's display name.</span></span>
