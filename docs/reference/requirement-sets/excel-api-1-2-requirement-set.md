# <a name="whats-new-in-excel-javascript-api-12"></a>Новые возможности API JavaScript для Excel 1.2

В ExcelApi 1.2 добавлена поддержка фильтрации таблиц и доступа к встроенным Excel функциям.

## <a name="api-list"></a>Список API

Чтобы просмотреть справочную документацию API для всех API, поддерживаемых Excel API JavaScript, за набором 1.2 или более ранних, см. в Excel API в наборе требований [1.2](/javascript/api/excel?view=excel-js-1.2&preserve-view=true)или ранее .
