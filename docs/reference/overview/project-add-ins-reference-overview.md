# <a name="javascript-api-for-project"></a>API JavaScript для Project

Project поддерживает надстройки, созданные с помощью API JavaScript, но в настоящее время отсутствует API JavaScript, предназначенный специально для взаимодействия с Project. Можно использовать [общий API](/javascript/api/office) для создания надстроек Project. Сведения об общем API см. в статье [API JavaScript для Office](../javascript-api-for-office.md). 

## <a name="learn-about-api-capabilities"></a>Сведения о возможностях API

Чтобы непосредственно использовать общий API для взаимодействия с содержимым в Project, выполните [краткие инструкции по началу работы с надстройкой Project](../../quickstarts/project-quickstart.md).
