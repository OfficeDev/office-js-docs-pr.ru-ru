# <a name="office-javascript-api-error-codes"></a>Коды ошибок API JavaScript для Office

В этой статье описываются сообщения об ошибках, которые могут возникать при использовании API JavaScript для Office (Office. js).

**Область применения**: надстройки Office | надстройки SharePoint | Excel | Outlook | PowerPoint | Project | Word

## <a name="binding-creation-error-conditions"></a>Условия ошибок создания привязки

При создании в API привязки укажите тип привязки, который вы хотите использовать. В следующих таблицах перечислены типы привязок и ожидаемое связанное с ними поведение.

### <a name="behavior-in-excel"></a>Поведение в Excel

В таблице ниже описано поведение привязок в Excel.

|**Указанный тип привязки**|**Выделенный фрагмент**|**Поведение**|
|:-----|:-----|:-----|
|Матрица|Диапазон ячеек (в том числе части таблиц и отдельные ячейки)|Привязка типа _матрица_ создается для выделенных ячеек. В документе не ожидается каких-либо изменений.|
|Матрица|Текст, выбранный в ячейке|Создается привязка типа _матрица_ для всей ячейки. В документе не ожидается каких-либо изменений.|
|Матрица|Несколько фрагментов или недопустимое выделение (например, пользователь выделяет изображение, объект или Word Art)|Создать привязку невозможно.|
|Table|Диапазон ячеек (в том числе отдельные ячейки)|Создать привязку невозможно.|
|Table|Диапазон ячеек в таблице (в том числе, отдельные ячейки таблицы, вся таблица или текст внутри ячейки таблицы)|Привязка создается для всей таблицы.|
|Table|Частичное выделение в таблице и за ее пределами|Создать привязку невозможно.|
|Table|Текст в ячейке (не в таблице)|Создать привязку невозможно.|
|Table|Несколько фрагментов или недопустимое выделение (например, пользователь выделяет изображение, объект, Word Art и т. д.)|Создать привязку невозможно.|
|Текст|Диапазон ячеек|Создать привязку невозможно.|
|Текст|Диапазон ячеек в таблице|Создать привязку невозможно.|
|Текст|Одна ячейка|Создается привязка типа  _текст_.|
|Текст|Одна ячейка в таблице|Создается привязка типа  _текст_.|
|Текст|Текст в ячейке|Создается привязка типа  _текст_ для всей ячейки.|

|**Указанный тип привязки**|**Выделенный фрагмент**|**Поведение**|
