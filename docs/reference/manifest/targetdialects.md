# <a name="targetdialects-element"></a>Элемент TargetDialects

Определяет региональные языки, которые поддерживает этот словарь.

**Тип надстройки:** надстройки области задач

## <a name="syntax"></a>Синтаксис

```XML
<TargetDialects>
   ...
</TargetDialects>
```
