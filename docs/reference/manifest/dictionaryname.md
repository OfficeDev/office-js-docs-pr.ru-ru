# <a name="dictionaryname-element"></a>Элемент DictionaryName

Указывает имя данного словаря.

**Тип надстройки:** надстройки области задач

## <a name="syntax"></a>Синтаксис

```XML
<DictionaryName DefaultValue="string" />
```
