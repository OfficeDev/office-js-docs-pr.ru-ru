# <a name="methods-element"></a>Элемент Methods

Указывает список методов из API JavaScript для Office, необходимых для активации надстройки Office.

**Тип надстройки:** контентные надстройки и надстройки области задач

## <a name="syntax"></a>Синтаксис

```XML
<Methods>
   ...
</Methods>
```

## <a name="contained-in"></a>Содержится в

[Requirements](requirements.md)

## <a name="can-contain"></a>Может содержать

[Метод](method.md)

## <a name="remarks"></a>Примечания

Элементы **Methods** и **Method** не поддерживаются в почтовых надстройках. Дополнительные сведения о наборах обязательных элементов см. в статье [Версии Office и наборы обязательных элементов](https://docs.microsoft.com/office/dev/add-ins/develop/office-versions-and-requirement-sets).
