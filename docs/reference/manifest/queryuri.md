# <a name="queryuri-element"></a>Элемент QueryUri

Указывает URL-адрес конечной точки для службы запросов словаря.

**Тип надстройки:** область задач

## <a name="syntax"></a>Синтаксис

```XML
<QueryUri DefaultValue="string" />
```
