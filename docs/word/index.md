# <a name="word-add-ins-documentation"></a>Документация по надстройкам Word

С помощью надстроек Word можно использовать знакомые веб-технологии, такие как HTML, CSS и JavaScript, для создания решения, работающего в Word на различных платформах, включая Windows, Mac, iPad и веб-браузер. Узнайте, как создавать, тестировать, отлаживать и публиковать надстройки Word.
