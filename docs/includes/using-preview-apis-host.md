> [!NOTE]
> API предварительной версии могут быть изменены и не предназначены для использования в рабочей среде. Рекомендуется использовать их только в тестовой среде и среде разработки. Не используйте API предварительной версии в рабочей среде или в важных деловых документах.
>
> Чтобы использовать API предварительного просмотра:
>
> - Вы должны ссылаться **на бета-библиотеку** на CDN ( https://appsforoffice.microsoft.com/lib/beta/hosted/office.js) . Файл [определения типа](https://appsforoffice.microsoft.com/lib/beta/hosted/office.d.ts) для компиляции и IntelliSense typeScript CDN и [DefinitelyTyped](https://raw.githubusercontent.com/DefinitelyTyped/DefinitelyTyped/master/types/office-js-preview/index.d.ts). Эти типы можно установить с `npm install --save-dev @types/office-js-preview` помощью .
> - Возможно, вам потребуется присоединиться к [программе Office insider](https://insider.office.com) для доступа к более последним Office сборкам.
