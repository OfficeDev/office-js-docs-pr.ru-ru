# <a name="excel-add-ins-documentation"></a>Документация по надстройкам Excel

С помощью надстроек Excel можно использовать знакомые веб-технологии, такие как HTML, CSS и JavaScript, для создания решения, работающего в Excel на различных платформах, включая Windows, Mac, iPad и веб-браузер. Узнайте, как создавать, тестировать, отлаживать и публиковать надстройки Excel.
