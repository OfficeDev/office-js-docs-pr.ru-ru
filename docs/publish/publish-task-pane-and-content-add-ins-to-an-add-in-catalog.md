# <a name="publish-task-pane-and-content-add-ins-to-a-sharepoint-app-catalog"></a><span data-ttu-id="b3bdb-103">Публикация надстроек области задач и контентных надстроек в каталоге приложений SharePoint</span><span class="sxs-lookup"><span data-stu-id="b3bdb-103">Publish task pane and content add-ins to a SharePoint app catalog</span></span>

<span data-ttu-id="b3bdb-p101">Каталог приложений — это отдельное семейство веб-сайтов в веб-приложении SharePoint или клиенте SharePoint Online, в котором размещены библиотеки документов для надстроек Office и SharePoint. Администраторы могут отправлять в него файлы манифестов надстроек Office, чтобы пользователи в организации могли получить доступ к этим надстройкам. When an administrator registers an app catalog as a trusted catalog, users can insert the add-in from the insertion UI in an Office client application.</span></span>
