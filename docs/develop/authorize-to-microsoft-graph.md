4. <span data-ttu-id="3cfee-122">Ведущее приложение Office запрашивает **маркер доступа к начальной загрузке** у конечной точки Azure AD версии 2.0 для текущего пользователя.</span><span class="sxs-lookup"><span data-stu-id="3cfee-122">The Office host application requests the **bootstrap access token** from the Azure AD v2.0 endpoint for the current user.</span></span>

10. <span data-ttu-id="3cfee-128">Azure AD возвращает надстройке маркер доступа в Microsoft Graph (и маркер обновления, если надстройка запрашивает разрешение *offline_access*).</span><span class="sxs-lookup"><span data-stu-id="3cfee-128">Azure AD returns the access token to Microsoft Graph (and a refresh token, if the add-in requests *offline_access* permission) to the add-in.</span></span>
