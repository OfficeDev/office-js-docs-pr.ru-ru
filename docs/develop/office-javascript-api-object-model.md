#### <a name="data-type-coercion"></a><span data-ttu-id="a246b-164">Приведение типов данных</span><span class="sxs-lookup"><span data-stu-id="a246b-164">Data type coercion</span></span>

Regardless of the actual shape of the binding, the different Office applications support the common data types by trying to coerce the data into the requested data type.
